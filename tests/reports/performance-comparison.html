Recomenda-se investigação imediata — qualquer slowdown adicional impacta diretamente o carregamento da lista de chats.
⚠️
Reclamações de lentidão no legado são válidas — o WebSocket não é instantâneo para a lista inicial. O handshake + entrega inicial leva 1.8s–5.6s após o load event. O p95 de 6.5s é perceptível pelo usuário.
✅
Nova interface mais rápida nos endpoints equivalentes — settings nova p95=392ms vs legado p95=939ms. O legado apresentou alta variância em 18/06 (p50 rápido mas p95 alto), sugerindo instabilidade na infra legada.
ℹ️
Auth agora mais rápido na nova interface — Foundation API p95=436ms vs Heimdall p95=797ms (18/06). Legado apresentou alta variância de p95. Ambos são imperceptíveis para o usuário mas o legado mostra tendência de degradação.
